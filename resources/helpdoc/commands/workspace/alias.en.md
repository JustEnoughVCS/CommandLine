> Manage the `ID Aliases` for the _local workspace_

## Usage
jvn workspace alias <parameter: ?>
__  --insert <local_id>  # Map local ID 
__  --to     <remote_id> # to remote ID

__  --erase  <local_id>  # Delete mapping for local ID
__  --query  <local_id>  # Query remote ID for local ID

## Aliases
__   -i <local_id> --to <remote_id>
__   -q <local_id>
__   -e <local_id>
